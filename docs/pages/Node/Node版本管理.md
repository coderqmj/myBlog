#### 1.首先需要安装n模块

```bash
sudo npm install -g n
```

#### 2.使用n模块对node.js版本的切换

```bash
# 后面放心填，即使本地没有也会帮你下载的
sudo n 10.0.0
```

#### 3.升级到node.js最新稳定版本

```bash
sudo n stable
```

#### 4.升级到最新版

```bash
sudo n latest
```

#### 5.n自行执行以查看下载的版本，并安装所选的版本

```bash
# 查看已经下载的版本号，可以通过键盘上的上下键进行切换。选择需要切换的版本回车即可
n
```

#### 6.删除指定版本

```bash
sudo n rm 7.10.0
```

#### 7.用指定的版本执行脚本

```
n use 10.0.0 index.js
```
