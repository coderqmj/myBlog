### 3.JS字符串前面后面补齐长度

```js
例子：

```

### 4.a标签锚点

- 很多时候会使用到a标签锚点，它是必须放文字内容的
- 但是很多时候是放一个背景图片即可，加了文字反而不是预期效果
- 但是不加文字影响SEO效果
- 解决办法就是`text-indent`属性

```
// 解决办法
text-indent: -9999px;
```
